# 2.css实现垂直居中的方法
 - 1. 绝对定位
   - 已知宽高 :width:200px;left:200px;
    .box{
        position:absolute;
        top:50%;
        left:50%;
        margin-left:-100px;
        margin-top: -100px;
    }
    或 calc ie10+
    .box{
        position:absolute;
        top:calc(50%-100px);
        left:calc(50%-100px);
    }
   未知宽高
   .box{
        position:absolute;
        top:50%;
        left:50%;
        transform: translate(-50%, -50%);
    }

# 3. 盒子宽未知，实现宽高比2：1
固定宽高比，宽度是百分比
.box{
    width:50vw;
    height:cal(50vw/2)
    <!-- 或 -->
    padding-top:calc(50vw*1/2)
    <!-- 或网格布局 -->
}
